# Relatório de Desenvolvimento: Aplicação Android 📱

Este repositório contém o desenvolvimento de uma aplicação Android interativa, focada na gestão de recursos visuais, interface adaptável e componentes de calendário.

---

## 🚀 Funcionalidades Implementadas

### 1. Gestão de Imagens (ImageView)
- Importação de recursos externos para a pasta `res/drawable`.
- Implementação de `ImageView` com boas práticas de acessibilidade utilizando `android:contentDescription`.

### 2. Interface de Calendário (CalendarView)
- Adição de um componente `CalendarView` para seleção de datas.
- Ajuste de restrições (Constraints) para garantir a visibilidade em diferentes tamanhos de ecrã.

### 3. Adaptabilidade (Landscape Layout)
- Criação de uma variante de layout para modo horizontal (`layout-land/activity_main.xml`).
- Reorganização dos elementos para otimizar o espaço em ecrãs largos.

### 4. Identidade Visual
- Configuração de um ícone de lançamento personalizado (Launcher Icon) utilizando o **Image Asset Studio**.

---

## 🛠️ Tecnologias e Ferramentas
- **Linguagem:** Java / Kotlin (escolhe a que usaste)
- **IDE:** Android Studio
- **Layout:** ConstraintLayout
- **Debug:** Logcat para monitorização de eventos e erros.
